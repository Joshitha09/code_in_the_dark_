Code in the Dark

How to Play

  • Memorize the buggy code which will only be shown for 20 seconds
  • You cannot change the language once the game is started
  • After the code disappears, type the corrected LINE only
  • You can reveal the code 3 times per question if needed
  • Score 20 points for each correct answer
  • Lose 2 points for each wrong attempt
  • Complete all 3 questions to finish the game
  • Overall time will be calculated for evaluation
  • You cannot exit from fullscreen if you do so you will be disqualified
